# Makarand Mane — Personal Brand Website

Premium, responsive, single-page personal brand website in **plain HTML + CSS** (no JS required).

## Folder structure

```
/website
  index.html
  styles.css
  /images
    makarand-hero.jpg
```

## Stack

- HTML5 + CSS3 only
- No React, no Tailwind, no Bootstrap, no PHP
- CSS-only mobile menu (checkbox hack)
- Fonts: Fraunces (serif headlines) + Inter (sans body) — editorial-tech hybrid
- Fully responsive, sticky header, smooth scroll, subtle fade-up animations

## Sections

1. Header / Navigation
2. Hero (with portrait on the right)
3. About
4. Services
5. Blog preview
6. Contact (with mailto form)
7. Footer

## Local preview

Just open `index.html` in any modern browser.

## Deploy

Drop these files into any static host (Netlify, Vercel, GitHub Pages, Cloudflare Pages, S3, etc).

## Customize

Open `styles.css` — all design tokens (colors, radii, spacing, fonts) are at the top under `:root`.
