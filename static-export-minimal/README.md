# Makarand Mane — Minimal Personal Site

Essay-style, single-column personal site in **plain HTML + CSS**.
Inspired by [abhaykulkarni.com](https://abhaykulkarni.com/) and [vallabhchitnis.com](https://www.vallabhchitnis.com/).

## Folder structure

```
/website
  index.html
  styles.css
  /images
    makarand-hero.jpg
```

## What's different from the premium version

- Single narrow column (~680px) — reads like a designed essay
- No cards, no shadows, no hero illustration — just type and rules
- Small circular avatar instead of a large hero portrait
- Conversational first-person voice throughout
- Toolkit as inline pills; experience/community as plain lists
- Honest, minimal — Vallabh/Abhay style

## Stack

- HTML5 + CSS3 only
- No JavaScript at all
- Fonts: Fraunces (serif) + Inter (sans)
- Fully responsive
- Smooth scroll, accessible focus states

## Deploy

Upload to any static host — Netlify, GitHub Pages, Vercel, Cloudflare Pages, S3.
